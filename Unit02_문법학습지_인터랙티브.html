텔레비전 앞으로 · ~는 중이다 & ~(으)ㄴ/는 대신(에)
총 점수 0 / 0
A-①
설명 · Explanation
먼저 설명을 읽고 이해해 보세요.
🇺🇸English
~는 중이다 is attached to a verb stem to indicate that an action is currently in progress.
It is similar to the English "-ing" form — something is happening right now and is not yet finished.

⚠️ Note: Only use with action verbs (동사). Do NOT use with adjectives (형용사).
🇰🇷한국어
~는 중이다는 동사 뒤에 붙어서 지금 어떤 동작이 진행 중임을 나타내요.
영어의 "-ing"처럼, 아직 끝나지 않고 계속되고 있는 동작에 사용해요.

📌 구조: 동사 어간 + 는 중이다 / 는 중이에요 / 는 중이야

설명 · Explanation
먼저 설명을 읽고 이해해 보세요.
🇺🇸English
~(으)ㄴ/는 대신(에) is used to express a substitution or contrast — doing one thing instead of another, or getting something in return for something else.
Think of it as "instead of…" or "in place of…"

It can be attached to nouns, verbs, or adjectives.
🇰🇷한국어
~(으)ㄴ/는 대신(에)는 앞에 나온 것 대신 다른 것을 하거나, 무언가를 대체하거나 교환할 때 사용해요.

📌 구조:
  • 동사 현재: 동사 어간 + 는 대신(에)
  • 동사 과거: 동사 어간 + (으)ㄴ 대신(에)
  • 명사: 명사 + 대신(에)
✅ 현재형 (Present)
텔레비전을 보는 대신에 산책했어요.
→ Instead of watching TV, I took a walk.
✅ 명사형 (Noun)
선물 대신에 용돈을 주세요.
→ Please give me allowance instead of a gift.
